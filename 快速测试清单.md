## 🧪 测试长沙完整流程

### 步骤1: 输入地址
- [ ] 点击"长沙湘江中路"快捷按钮
- [ ] 确认输入框中显示"长沙湘江中路"

### 步骤2: 查看区域分析报告
- [ ] 等待3秒加载动画
- [ ] 右侧文档面板自动打开
- [ ] 显示"长沙湘江中路区域分析报告"
- [ ] 报告包含以下部分:
  - [ ] 一、历史文化调研
  - [ ] 二、人文底蕴调研
  - [ ] 三、社区人文调研
  - [ ] 四、周边酒店竞品调研
  - [ ] 五、周边景点调研
- [ ] 点击"确认通过"按钮

### 步骤3: 查看故事主题
- [ ] 左侧显示8个主题卡片
- [ ] 每个卡片包含:主题名、副标题、核心元素、描述
- [ ] 8个主题分别是:
  - [ ] 枫声夜语
  - [ ] 青砖呼吸录
  - [ ] 雾江光书
  - [ ] 舟语茶韵
  - [ ] 声墙迷径
  - [ ] 桥洞星野
  - [ ] 渔火星辞
  - [ ] 碑影沉香

### 步骤4: 选择3个主题
- [ ] 点击选择"枫声夜语"(卡片高亮显示)
- [ ] 点击选择"青砖呼吸录"
- [ ] 点击选择"雾江光书"
- [ ] 确认显示"已选择 3 / 3 个主题"
- [ ] 出现"生成邻间故事文档"按钮

### 步骤5: 生成邻间故事文档
- [ ] 点击"生成邻间故事文档"按钮
- [ ] 右侧文档面板显示邻间故事文档
- [ ] 文档标题: "长沙湘江中路邻间故事设计文档"
- [ ] 显示3个已选主题卡片

### 步骤6: 验证文档内容
#### 6.1 综合主线故事(默认显示)
- [ ] 点击"综合主线故事"导航按钮(应该已高亮)
- [ ] 显示标题"融合 枫声夜语、青砖呼吸录、雾江光书 三个主题的核心叙事"
- [ ] 显示完整的综合故事内容(约500-600字)
- [ ] 内容融合了3个主题的元素

#### 6.2 主题展开故事
- [ ] 点击"主题展开故事"导航按钮
- [ ] 显示3个主题的详细故事
- [ ] 每个主题包含:
  - [ ] 主题序号(1/2/3)
  - [ ] 主题名称和副标题
  - [ ] 完整的展开故事内容(约400-500字)
- [ ] 确认3个主题都有内容显示

#### 6.3 酒店设计灵感
- [ ] 点击"酒店设计灵感"导航按钮
- [ ] 显示3个主题的设计方案
- [ ] 每个主题包含13个设计维度:
  - [ ] 1. 外部建筑
  - [ ] 2. 室内建筑
  - [ ] 3. 材料与饰面
  - [ ] 4. 艺术品
  - [ ] 5. 雕塑与装置艺术
  - [ ] 6. 配饰
  - [ ] 7. 家具
  - [ ] 8. 照明
  - [ ] 9. 软装配饰
  - [ ] 10. 餐饮服务
  - [ ] 11. 服务用品
  - [ ] 12. 香氛
  - [ ] 13. 声音
